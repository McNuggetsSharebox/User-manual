# Customer Focus 内测用户手册

## 1. 文档目的

本手册面向参与部门内测的同事，用于帮助大家快速了解 Customer Focus 的当前能力、使用方式和已知边界。

当前版本定位为内部试用版，重点验证以下主链路：

- Cloudreve 账号登录
- 文件上传与分类管理
- AI 摘要生成与查看
- 基于知识库的问答
- 文件与目录操作事件留痕
- Cloudreve 云盘入口打通

请将本系统视为内部知识处理工具，当前输出仅供参考，不作为正式业务结论、法律意见或商务决策依据。

## 2. 系统简介

Customer Focus 是一个面向业务资料的 AI 摘要与问答工作台，当前围绕四类资料开展处理：

- 客户
- 供应商
- 竞争对手
- 西门子

系统当前由两个部分组成：

- Dashboard：用于查看摘要、管理文件、生成摘要、使用 AI Bot
- Cloudreve：用于账号登录和进入云盘查看文件

## 3. 使用前准备

开始使用前，请确认以下事项：

1. 已获得内测访问地址。
2. 已有可用的 Cloudreve 账号和密码。
3. 使用现代浏览器访问，建议优先使用 Edge 或 Chrome。
4. 浏览器允许弹出新窗口，否则“进入云盘”可能无法正常打开。

## 4. 登录系统

### 4.1 登录步骤

1. 打开内测地址。
2. 在登录页输入 Cloudreve 邮箱。
3. 输入 Cloudreve 密码。
4. 点击“登录并进入 Dashboard”。

登录成功后会直接进入 Dashboard。后续点击“进入云盘”时，通常无需再次输入账号密码。

### 4.2 登录失败时优先检查

- 邮箱或密码是否输入错误
- Cloudreve 账号是否可正常使用
- 当前网络是否可访问内网部署环境

## 5. 首页 Dashboard

登录后默认进入 Dashboard 页面。页面主要由以下几部分组成。

### 5.1 顶部区域

顶部会显示：

- 当前登录人信息
- Dashboard / 文件管理 页面切换按钮
- 事件页面入口（仅 admin / owner 可见）
- 刷新摘要
- 退出账号
- 进入云盘

同时会展示当前摘要数量，包括：

- 总摘要数
- 客户摘要数
- 供应商摘要数
- 竞争对手摘要数
- 西门子摘要数

### 5.2 搜索摘要

Dashboard 顶部提供“搜索文件”输入框，可按以下信息进行筛选：

- 文件名
- 路径
- 分类名称

该搜索用于快速定位当前已加载的摘要卡片，不是全文语义检索。

### 5.3 四个摘要栏目

Dashboard 当前固定为四个栏目：

- 客户
- 供应商
- 竞争对手
- 西门子

每个栏目展示对应分类下的摘要内容。摘要卡片中通常可看到：

- 文件名
- 更新时间
- 摘要标签
- 摘要预览
- “展开全文”按钮
- “打开源文件”链接

如需阅读完整摘要，可点击“展开全文”。

### 5.4 聚焦某个栏目

点击某个栏目空白区域后，可让该栏目进入聚焦状态，便于集中查看；再次点击可取消聚焦。

### 5.5 删除摘要

当前登录用户可在摘要卡片上执行“删除摘要”。

请注意：

- 删除的是 Dashboard 中的摘要记录
- 不会删除 Cloudreve 中的原始文件
- 当前删除摘要后，相关 RAG 检索内容可能仍然保留
- 该操作会写入事件日志

## 6. 进入云盘

点击顶部“进入云盘”按钮后，系统会新开页面进入 Cloudreve。

适用场景：

- 查看云盘中的原始文件
- 继续在 Cloudreve 中做文件协作
- 核对目录结构和文件实际位置

如果点击后没有打开新页面，通常是浏览器拦截了弹窗，请允许该站点弹出窗口后重试。

## 7. 文件管理页面

点击顶部“文件管理”可进入文件管理页。该页面包含两部分：

- 上传文件面板
- 文件管理列表

### 7.1 上传文件

在“上传文件”区域，您可以按以下步骤操作：

1. 选择目标分类。
2. 在目录树中选择 Cloudreve 目标目录。
3. 选择本地文件。
4. 视文件类型决定是否勾选“上传后生成摘要”。
5. 点击“上传文件”或“上传并生成总结”。

### 7.2 支持的文件类型

当前文件处理能力如下：

可直接生成摘要：

- DOCX
- PDF
- TXT
- MD

可进入 RAG 检索：

- DOCX
- PDF
- TXT
- MD
- CSV

说明：

- 如果文件类型支持摘要，页面会允许勾选“上传后生成摘要”。
- 如果文件类型不支持摘要，该勾选项会自动禁用。
- 不是所有上传文件都会自动生成摘要，但部分文件即使没有摘要，仍可进入问答检索。

### 7.3 上传结果说明

上传后可能出现以下结果：

- 上传并生成摘要：文件完成上传，摘要立即生成，并同步进入 Dashboard
- 仅上传文件：文件完成上传，若文件类型可索引，仍会进入 RAG，后续可手动生成摘要

### 7.4 上传大小限制

当前后端默认单文件上传上限为 100 MB。

如果超出限制，系统会提示上传失败，并返回明确的大小限制错误。若后续部署配置调整，以页面实际提示为准。

### 7.5 目录管理

当前登录用户可在目录树中执行：

- 新建目录
- 重命名目录
- 删除目录

说明：

- 目录操作直接作用于 Cloudreve
- 目录操作会写入事件日志
- “事件”页面仅对 admin / owner 可见

## 8. 文件管理列表

上传区域下方为“文件管理”列表，用于查看当前受管文件状态。

### 8.1 列表中可看到的信息

- 文件所属分类
- 文件名
- Cloudreve 路径
- 上传时间
- 摘要状态

### 8.2 摘要状态说明

当前状态分为三类：

- 已有摘要
- 摘要已删除
- 待生成

含义如下：

- 已有摘要：该文件已在 Dashboard 中生成摘要
- 摘要已删除：摘要文本已从 Dashboard 移除，但相关 RAG 检索仍可能保留，可重新生成摘要
- 待生成：文件已存在于受管目录中，当前尚未生成摘要

### 8.3 手动生成或重新生成摘要

在文件管理列表中，可对文件点击：

- “生成摘要”
- “重新生成摘要”

适用场景：

- 上传时未勾选“上传后生成摘要”
- 已删除摘要后需要恢复
- 希望基于当前文件内容重新生成摘要

### 8.4 删除摘要

当前登录用户可在文件管理页删除摘要。

删除摘要后：

- Dashboard 中该文件的摘要卡片将消失
- Cloudreve 中原始文件仍保留
- 如有需要，可后续重新生成摘要
- 该操作会写入事件日志

### 8.5 删除文件

文件管理页当前提供“删除文件”按钮。

删除文件后将同步执行以下动作：

- 删除 Cloudreve 中的原始文件
- 删除 Dashboard 中对应摘要记录
- 删除 PostgreSQL 中对应 RAG 索引
- 写入事件日志

该操作不可撤销。点击“删除文件”后，系统会弹出确认窗口，请确认文件不再需要后再继续。

### 8.6 文件管理状态筛选

文件管理页顶部支持按状态快速筛选：

- 已有摘要
- 待生成 / 已删除

也支持通过搜索框按文件名、路径、分类快速过滤当前列表。

## 9. 事件页面

如果当前账号属于 admin 或 owner，顶部会显示“事件”入口。

### 9.1 事件页用途

事件页用于查看关键管理操作的留痕记录，当前包括：

- 删除摘要
- 删除文件
- 新建目录
- 重命名目录
- 删除目录

### 9.2 事件页中可看到的信息

- 操作时间
- 操作人
- 操作目标
- 目标路径
- 操作类别
- 补充说明

## 10. AI Bot 使用说明

页面右下角提供 AI Bot 浮窗。

### 10.1 打开方式

点击右下角机器人按钮即可打开问答窗口。

### 10.2 提问方式

- 直接输入问题
- 按 Enter 发送
- 按 Shift + Enter 换行

建议优先提与已上传资料相关的问题，例如：

- 某客户资料中提到的核心需求是什么？
- 某供应商文件中有哪些风险点？
- 某竞品材料里提到了哪些产品特点？
- 某份西门子内部文件的重点结论是什么？

### 10.3 回答类型

Bot 当前有两种回答来源：

- 基于已上传文件的检索增强回答
- 通用大模型回答

如果系统在知识库中检索到相关内容，回答中通常会带上：

- 引用文件名
- 所属分类
- 命中片段
- 可访问链接

如果系统没有检索到足够证据，可能会回退到通用回答，并明确说明该结论不一定来自已上传文档。

### 10.4 使用建议

- 尽量把问题问具体，不要过于泛化
- 问题中可以带文件主题、分类或背景词
- 对关键结论，建议再点击源文件链接进行人工核对

## 11. 常见使用场景

### 11.1 想找某一份文件，但只记得主题或内容

推荐做法：

1. 先打开 AI Bot，输入您记得的关键信息，例如客户名、项目名、产品名、会议主题。
2. 从 Bot 返回的结果里记录文件名、引用片段，必要时记录对应文件号或路径关键词。
3. 回到 Dashboard 顶部搜索框，或进入“文件管理”页的搜索框，输入文件名、文件号、路径关键词继续定位。
4. 定位后可直接打开源文件，或在文件管理页继续执行生成摘要、删除摘要、删除文件等操作。

适用示例：

- “帮我找到提过客户 A 需求变更的那份文件”
- “上次关于某竞品报价的材料是哪一份”

### 11.2 想验证 Bot 的回答是不是有依据

推荐做法：

1. 先在 Bot 中提问。
2. 查看回答下方的引用文件名、命中片段和链接。
3. 点击“打开源文件”或引用链接，直接回看 Cloudreve 中的原始文件。
4. 重要结论请以源文件为准，不要只看 Bot 的整理结果。

### 11.3 文件已经上传，但还没有摘要

推荐做法：

1. 进入“文件管理”页面。
2. 用搜索框按文件名、路径或分类定位该文件。
3. 如果状态是“待生成”或“摘要已删除”，点击“生成摘要”或“重新生成摘要”。
4. 生成完成后，再回到 Dashboard 查看摘要卡片是否出现。

### 11.4 想彻底移除某份错误或过期文件

推荐做法：

1. 进入“文件管理”页面并搜索到目标文件。
2. 如果只是不想展示摘要，可点击“删除摘要”。
3. 如果要连同 Cloudreve 原文件一起清理，请点击“删除文件”。
4. 删除文件后，系统会同步删除 Cloudreve 文件、摘要记录、RAG 索引，并写入事件日志。

## 12. 推荐使用流程

对于首次内测的同事，建议按以下顺序体验：

1. 使用 Cloudreve 账号登录系统。
2. 在 Dashboard 浏览四类摘要内容。
3. 点击“进入云盘”，确认云盘入口可正常打开。
4. 进入“文件管理”，上传一份测试文件。
5. 如支持摘要，可勾选“上传后生成摘要”。
6. 上传完成后返回 Dashboard 查看新摘要是否出现。
7. 进入“文件管理”查看该文件状态，并按需生成摘要、删除摘要或删除文件。
8. 如具备 admin / owner 权限，可进入“事件”页查看操作留痕。
9. 打开 AI Bot，围绕刚上传的文件提问。

## 13. 已知边界与注意事项

当前内测版本请特别注意以下几点：

1. 系统仍处于内部试用阶段，结果仅供参考。
2. 摘要生成能力目前只覆盖 DOCX、PDF、TXT、MD。
3. CSV 当前可进入 RAG 检索，但不会直接生成摘要。
4. 删除摘要不会删除云盘原文件。
5. 删除文件会直接删除 Cloudreve 原文件，请谨慎操作。
6. 文件变更同步可能存在短暂延迟，必要时可手动点击“刷新摘要”。
7. 问答结果依赖当前知识库内容，若文档未被成功索引，Bot 可能无法引用到它。

## 14. 常见问题

### 14.1 登录后反复回到登录页

请先确认：

- Cloudreve 账号密码正确
- 浏览器未禁用 Cookie
- 当前网络可访问部署环境

### 14.2 点击“进入云盘”没有反应

通常是浏览器拦截了新窗口。请允许该站点的弹窗权限后再试。

### 14.3 上传文件后不能勾选“上传后生成摘要”

说明当前文件类型不在摘要支持范围内。若文件属于可索引类型，仍可能进入 RAG 检索。

### 14.4 上传成功后没有马上看到摘要

可先尝试：

- 点击“刷新摘要”
- 切换到“文件管理”查看该文件状态
- 确认上传时是否选择了“上传后生成摘要”
- 对文件手动点击“生成摘要”

### 14.5 为什么 Bot 的回答和文件内容不完全一致

Bot 会基于检索片段和大模型能力进行整理，不等于逐句复述原文。对重要信息请始终回到源文件进行核实。
